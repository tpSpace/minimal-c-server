# Minimal C server

## Read the manual
```bash
man socket
man bind
man sockaddr_in
man lsiten
man accept
man recv
man strchr
man 2 open
```
## Compile code
```bash
gcc server.c -o server
```
## Run server
```bash
./server
```
## Make a request
```bash
wget localhost:8080/index.html
```
- Better to run it in /tmp directory.
